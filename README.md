# Lumos Legacy

A magical Hogwarts-inspired React + Vite experience with a cinematic hero section, interactive Sorting Hat ceremony, animated house cards, and custom house welcome pages.

## Live Site

https://lumoslegacy.netlify.app/

## Features

- Scroll-based Hogwarts hero video
- Magical Sorting Hat ceremony
- House reveal animation
- Gryffindor, Slytherin, Ravenclaw, and Hufflepuff house pages
- Animated particles, glowing effects, runes, lightning, and confetti
- Custom wand cursor and themed visuals

## Tech Stack

- React
- Vite
- GSAP
- CSS animations
- Netlify
