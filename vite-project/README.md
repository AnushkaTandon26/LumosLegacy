# Hogwarts Experience

An interactive React + Vite page with a scroll-scrubbed Hogwarts hero video, Sorting Hat reveal, animated house cards, particles, and themed cursor effects.

## Run locally

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Vite will print a local URL, usually `http://localhost:5173/`.

## Useful scripts

```bash
npm run lint
npm run build
npm run preview
```

## Notes

- The hero uses `public/video/one.mp4`.
- House crest images live in `public/images`.
- The selected house is saved in local storage under `hp-house`.
